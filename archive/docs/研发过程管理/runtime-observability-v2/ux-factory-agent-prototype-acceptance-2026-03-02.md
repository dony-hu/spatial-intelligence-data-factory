# 最小验收清单：Factory Agent 全新交互原型（2026-03-02）

## 1. 与工厂 Agent 交互
1. 页面存在对话输入框与发送动作。
2. 发送后对话区新增用户消息与 Agent 回复。

## 2. 可见 Agent 与 LLM 交互
1. 页面展示 Agent→LLM 请求与 LLM→Agent 响应。
2. 每轮至少展示模型、耗时、token 或摘要信息。

## 3. 可见工作包制定过程与结果
1. 页面展示工作包阶段流转（创建、收敛、确认、打包、试运行、发布）。
2. 页面展示工作包产物结果（id、version、规则摘要、发布状态）。

## 4. 可见治理输入到输出过程与结果
1. 页面可录入/粘贴输入样本。
2. 页面展示逐条处理阶段与最终输出结果。
3. 页面展示批次级结果摘要（成功率、阻断数、图谱构建状态）。

## 5. 工作包选择与维护来源
1. 治理执行前必须从工作包下拉选择 `workpackage_id@version`，不可自由输入。
2. 工作包列表显示“来源于 Agent↔LLM 自动维护”，并可看到最近维护记录。
3. Agent 新增对话轮次或手动触发维护后，工作包列表可实时刷新并默认选中最新版本。
