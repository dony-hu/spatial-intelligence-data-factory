# Story：OBS-RUNTIME-S2-3 地址治理样例包灌入与空态引导

## 1. 目标

提供可重复灌入的地址治理样例包，确保页面可直接看到治理成果，并具备空态引导能力。

## 2. 范围

1. 样例包生成与灌入脚本
2. 样例数据分层：
   - 高置信自动通过 20 条
   - 中置信需复核 20 条
   - 低置信高风险 10 条
   - 阻塞/失败 10 条
3. 页面空态引导（无数据时提示灌入）

## 3. 验收标准

1. 灌入后任务总数不少于 60。
2. 页面 KPI、风险分布、明细列表均非空。
3. 指标落在目标区间（完成率/阻塞率/人工介入率）。
4. 样例包可重复执行，结果分布稳定。
5. 灌入失败时返回显式错误，不允许 fallback。

## 4. 测试要求（TDD）

1. 先补失败用例：
   - 样例包契约测试
   - 灌入流程测试
   - 空态转非空测试
2. 再实现脚本与引导逻辑。
3. 最后执行端到端验收。

## 5. 依赖

1. S2-1 API 可写入/查询。
2. 任务与审核回写接口已稳定。
