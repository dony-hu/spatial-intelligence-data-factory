# Story：OBS-RUNTIME-S2-4 任务下钻追溯与回归验收闭环

## 1. 目标

建立任务级下钻追溯与全链路验收闭环，支持从页面定位问题到证据链复核。

## 2. 范围

1. 任务详情接口（输入/输出/strategy/evidence/review/audit）
2. 页面任务下钻面板
3. Trace 回放入口
4. 验收证据产出（JSON + Markdown）

## 3. 验收标准

1. 点击任一任务可查看完整证据链。
2. 审核记录与任务状态演进一致。
3. Trace 回放可按 `task_id` 关联关键事件。
4. 形成可复核验收报告并归档。
5. 关键查询失败显式报错，不做降级返回。

## 4. 测试要求（TDD）

1. 先补失败用例：
   - 任务下钻契约测试
   - 回放一致性测试
   - 审核一致性测试
2. 再实现详情与回放能力。
3. 最后回归并输出验收报告。

## 5. 依赖

1. `audit.events` 结构化事件可用。
2. S2-1/S2-2/S2-3 已完成。
