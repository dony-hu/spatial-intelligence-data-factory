# 研发主题 - 架构收口专项（2026-02-27）

## 1. 主题目标

围绕最新目标架构（`docs/02_总体架构/系统总览.md`、`docs/02_总体架构/模块边界.md`、`docs/02_总体架构/依赖关系.md`），完成从“可运行”到“架构一致”的收口，重点消除数据模型漂移、运行时 DDL、边界穿透与观测真相源不一致问题。

## 2. 核心约束

1. 严格 PG-only：不允许 SQLite 运行路径。
2. 严格 No-Fallback：关键链路失败必须 `blocked/error`。
3. Alembic 是唯一生产 DDL 入口。
4. 仅 Repository/DAO 可访问数据库。
