# 术语表

| 术语 | 含义 |
|---|---|
| Factory Agent | 负责目标对齐、工作包生成和门禁编排的上游编排器 |
| Workpackage | 可发布、可执行、可审计的治理交付单元 |
| Bundle | 工作包的可执行目录与脚本封装 |
| `workpackage_id@version` | 运行时识别并执行工作包的标准标识 |
| Trust Hub | 可信能力目录和可信数据查询层 |
| Dryrun | 发布前试运行与验证阶段 |
| `WAIT_USER_INPUT` | 需要用户补信息或做裁决 |
| `WAIT_USER_GATE` | 结果已具备，只等待用户门禁确认 |
| Evidence | 运行结果旁证，如审计、trace、产物摘要 |
| No-Fallback | 失败必须显式暴露，不能静默降级 |
